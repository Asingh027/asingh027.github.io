Algobot

Automatic Trading Algorithm that utilizes the Alpaca API to push buy and sell orders based on stock recommendations.

Introduction

The Algobot project is an automatic trading program that I developed shortly after graduating from Stevens Institute of Technology. My degree was in Quantitative Finance and I had prior exposure to automated trading in the past, so I found it quite interesting and decided to give it a try. Some time after beginning my Master's degree in Data Science, I rediscovered my source code and decided to revisit the project and see if I could improve the program's performance.

Source Code

The following section contains information about and links to the various source code files utilized in this project.

Algobot V1.0.py

This file was the original code I implemented in 2020, when I first started this project. The original code utilized the 5-day Moving Average and the 20-day Moving Average to indicate buy or sell signals, then pushed these evaluations as buy or sell orders via the Alpaca API. It re-evaluates the stock price every minute and sells if it detects a negative trend.

stock_price_trend_evaluator.py

This file contains a class indicators which takes in a list of tickers, fetches the historical price data, then calculates each selected indicator's value. Once these calculations are done, the class's execute function runs, which appends the final recommendation based on each signal to a pandas DataFrame object. The recommendation takes form as a True/False value, where True indicates Buy and False indicates Sell. As the model evolves and I conduct research on better trading signals, my strategy will continue to improve! The current indicators I am implementing are MACD, Bollinger Bands, and RSI.
